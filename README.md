# 🚀 Canadian Tech Job Market Analysis | Power BI Dashboard

## Project Overview
This project presents a comprehensive analysis of the **Canadian tech job market** using publicly available **LinkedIn job postings dataset**. The objective was to simulate a real-world analytics workflow, from raw dataset to decision-ready insights, focusing on **data cleaning, transformation, feature engineering, and interactive dashboard design**.  

The dashboard provides actionable insights into **location-based job distribution, experience requirements, skill demand trends, and role-level insights**, replicating the type of work expected in data analyst and business intelligence roles.  

## Dataset
- **Source:** [LinkedIn Job Postings on Kaggle](https://www.kaggle.com/datasets/arshkon/linkedin-job-postings)  
- **Content:** Job titles, company names, locations (provinces and cities in Canada), skills, job descriptions, job type, and experience requirements.  
- **Scope:** Filtered to include **data-related tech roles** across Canada (Data Analyst, Data Scientist, BI roles, etc.).  

## Key Tasks & Workflow
1. **Data Cleaning & Preprocessing**
   - Handled missing values, duplicates, and inconsistent entries.  
   - Standardized location fields (province and city) and role titles.  
   - Removed irrelevant or non-Canada job postings.  

2. **Feature Engineering**
   - Created experience buckets (e.g., Entry, Mid, Senior level).  
   - Parsed locations to province and city for geographic analysis.  
   - Extracted and standardized skills for skill demand analysis.  

3. **Power BI Dashboard Design**
   - **Province & city-level job distribution:** Heatmaps and charts showing concentration of tech roles.  
   - **Experience requirement analysis:** Visualizations by role and location.  
   - **Skill demand trends:** Commonly required skills across Canadian tech roles.  
   - **Interactive filters & slicers:** Role, province, city, and skill-level filtering.  
   - Applied **best practices in DAX, data modeling, and visual storytelling**.  

## Key Insights
- Provinces and cities with the highest concentration of tech roles in Canada.  
- Variation in experience requirements across different roles and locations.  
- Most commonly demanded skills across Canadian tech jobs.  
- Overview of the Canadian tech hiring landscape.  

## Tools & Technologies
- **Power BI:** Dashboard creation, DAX measures, data visualization.  
- **Python / Pandas:** Data cleaning, transformation, and feature engineering.  
- **Excel:** Initial exploration and validation of dataset.  

## How to Use
1. Download the [dataset](https://www.kaggle.com/datasets/arshkon/linkedin-job-postings).  
2. Open the Power BI file (`.pbix`) included in this repository.  
3. Explore the **interactive visuals** using slicers for roles, skills, provinces, and cities.  
4. Customize filters to analyze different data-related job trends across Canada.  

## Project Outcome
This project demonstrates the **end-to-end process of real-world analytics** — from raw LinkedIn job postings to an actionable, interactive Power BI dashboard — showcasing capabilities in **data cleaning, modeling, feature engineering, and visual storytelling**.
